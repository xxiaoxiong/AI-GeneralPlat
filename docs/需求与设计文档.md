# 国产化内网部署型「企业级 AI 通用能力大平台与业务落地平台」需求与设计文档

**版本**：v1.0  
**日期**：2026-02-22  
**状态**：草稿

---

## 一、产品定位与目标

### 1.1 产品定位

专为国企、事业单位、上市公司、中小微企业打造的**可完全私有化部署、国产化全栈适配、全业务场景可编排**的 AI 能力统一管控与落地平台。一站式解决企业"不敢用、不会用、用不起、管不好"AI 的核心痛点。

### 1.2 目标用户

| 用户类型 | 规模 | 核心诉求 |
|---|---|---|
| 国企 / 事业单位 | 大型 | 等保合规、国产化替代、数据不出内网 |
| 上市公司 | 中大型 | 数据安全、业务流程自动化、审计留痕 |
| 中小微企业 | 中小型 | 低成本快速落地、开箱即用 |
| 金融 / 能源 / 制造 / 政务 / 教育行业 | 各规模 | 行业专属场景、深度定制 |

### 1.3 核心价值主张

- **敢用**：完全内网离线，数据零外泄，满足等保三级、涉密合规
- **会用**：无代码可视化编排，非技术人员即可上手
- **用得起**：从几万元中小企业版到数百万定制版，覆盖全规模客户
- **管得好**：统一权限、用量、审计、计费，全链路可控

## 二、核心痛点与解决方案映射

| 痛点 | 现状 | 本平台解决方案 |
|---|---|---|
| 合规与数据安全 | 90% AI 工具为 SaaS，数据上传外网，违反等保 2.0 | 完全私有化部署，全程内网，数据不出企业 |
| 能力碎片化 | 对接 N 个厂商 API，账号/权限/数据割裂 | 统一多模型调度中枢，单平台管理所有 AI 能力 |
| 落地难 | 通用模型不懂业务，无行业知识，无技术团队 | 企业知识库 + 无代码编排引擎，非技术人员可落地 |
| 成本高 | 大厂私有化百万起步，定制开发天价 | 模块化订阅，按需付费，中小企业可承受 |

## 三、功能需求

### 3.1 多模型统一调度与管控中枢

**功能描述**：作为企业所有 AI 能力的统一入口，屏蔽底层模型差异。

**核心需求**：
- 支持对接国内外主流大模型 API（文心一言、通义千问、智谱 GLM、DeepSeek、百川、Kimi、GPT-4、Claude 等）
- 支持开源模型本地化部署（Llama、Qwen、ChatGLM、DeepSeek 等），通过 Ollama / vLLM / TGI 等推理框架接入
- 统一 API 网关：对上层应用提供标准化接口，屏蔽底层模型差异
- 智能路由：按任务类型、成本、响应速度自动选择最优模型
- 负载均衡与故障转移：多模型实例间自动调度，单点故障自动切换
- 用量计费：按 Token、按调用次数、按部门/用户维度统计与计费
- 权限管控：细粒度控制哪些用户/部门/应用可调用哪些模型
- 数据审计：所有调用请求与响应全量日志，可查询、可导出

**非功能需求**：
- API 响应延迟 P99 < 5s（本地模型）
- 支持并发调用 ≥ 100 路（可横向扩展）
- 审计日志保留 ≥ 180 天

### 3.2 国产化全栈适配

**功能描述**：确保平台在纯国产化软硬件环境下稳定运行。

**适配矩阵**：

| 层次 | 支持范围 |
|---|---|
| 国产 CPU | 鲲鹏（ARM）、飞腾（ARM）、海光（x86）、龙芯（MIPS/LoongArch） |
| 国产操作系统 | 统信 UOS、麒麟 KylinOS（银河麒麟 / 中标麒麟）、欧拉 openEuler |
| 国产数据库 | 达梦 DM8、人大金仓 KingbaseES、GaussDB、OceanBase、TiDB |
| 国产中间件 | 东方通 TongWeb、金蝶 Apusic、宝兰德 BES |
| 国产向量数据库 | 星环 Transwarp、自研适配 Milvus 国产化版本 |
| 国产 AI 推理框架 | 昇腾 CANN、寒武纪 MLU、天数智芯 |

**合规要求**：
- 满足等保 2.0 三级要求
- 支持涉密信息系统（分级保护）部署模式
- 支持完全内网离线运行，无任何外网依赖
- 支持国密算法（SM2/SM3/SM4）加密

### 3.3 无代码 AI 业务流程编排引擎

**功能描述**：可视化拖拽式流程设计器，非技术人员即可将 AI 能力嵌入业务流程。

**核心需求**：

**流程设计器**：
- 基于 Web 的可视化画布，拖拽式节点编排
- 内置节点类型：触发器、AI 处理节点、条件判断、数据转换、系统对接、人工审批、定时任务、消息通知
- 支持流程版本管理、发布、回滚
- 支持流程模板库（内置行业最佳实践模板）

**AI 节点能力**：
- 文本生成、摘要、翻译、分类、抽取、问答
- 文档解析（OCR、表格识别、图纸解析）
- 图像识别、视频分析
- 语音转文字、文字转语音

**系统对接节点**：
- OA 系统（泛微、蓝凌、致远）
- ERP 系统（SAP、用友、金蝶）
- CRM 系统（Salesforce、纷享销客）
- 财务系统（用友 NC、金蝶云星空）
- 自定义 HTTP/REST/SOAP 接口对接
- 数据库直连（MySQL、Oracle、达梦等）
- 消息队列（Kafka、RabbitMQ、RocketMQ）

**典型流程示例**：
> 合同上传 → AI 智能审查 → 风险预警 → 法务审批 → 电子签章 → 归档 → 财税对接

**非功能需求**：
- 单流程节点数支持 ≥ 200 个
- 流程执行历史可查询、可重放
- 支持并行分支与子流程调用

### 3.4 企业专属知识库与知识图谱引擎

**功能描述**：构建企业私有知识资产，解决大模型幻觉问题，实现精准可溯源的知识问答。

**文档处理能力**：
- 支持格式：Word、PDF、Excel、PPT、TXT、Markdown、HTML、图纸（DWG/DXF）、音频（MP3/WAV）、视频（MP4）
- 自动解析、分块、向量化、索引
- OCR 识别（支持国产 OCR 引擎）
- 表格结构化抽取
- 图片内容理解

**知识检索**：
- 混合检索：向量语义检索 + 关键词全文检索（BM25）融合排序
- 精准溯源：答案关联原始文档段落，支持一键跳转原文
- 多知识库隔离：按部门、项目、密级独立管理
- 知识库权限控制：细粒度读写权限

**知识图谱**：
- 自动实体抽取与关系构建
- 图谱可视化浏览
- 支持人工校正与补充
- 图谱与向量检索联合推理

**应用场景**：
- 内部培训与知识传承
- 智能客服与 FAQ 问答
- 辅助决策与政策查询
- 技术文档检索

### 3.6 全链路数据安全与审计

**功能描述**：覆盖数据全生命周期的安全防护与合规审计体系。

**数据安全**：
- 传输加密：TLS 1.3 + 国密 SM4
- 存储加密：数据库字段级加密，文件存储加密
- 敏感信息自动识别与脱敏（身份证、手机号、银行卡、密码等）
- 数据分级分类标注
- 数据不出内网：所有计算在本地完成，无外网调用

**访问控制**：
- RBAC 角色权限模型（角色 → 权限 → 资源）
- 支持对接企业 LDAP/AD 域账号
- 支持 SSO 单点登录（CAS、OAuth2、SAML）
- 多因素认证（MFA）
- IP 白名单与访问时段限制

**审计日志**：
- 用户登录/操作行为全量记录
- AI 对话内容全量留痕（含输入输出）
- 文件上传/下载/删除记录
- 系统配置变更记录
- 日志防篡改（哈希链）
- 支持日志导出与第三方 SIEM 对接

**合规报告**：
- 一键生成等保合规报告
- 数据使用统计报告
- 安全事件告警与通知

## 七、核心业务流程示例

### 7.1 智能合同审查流程

```
[用户上传合同 PDF]
       │
       ▼
[文档解析服务] → OCR + 结构化抽取
       │
       ▼
[AI 合同审查节点] → 调用大模型，结合法律知识库
       │
       ▼
[风险评分与标注] → 输出风险点列表（高/中/低）
       │
       ▼
[人工审批节点] → 法务人员在线审阅，标注意见
       │
       ├─ 通过 ──→ [电子签章] → [归档] → [财税系统对接]
       │
       └─ 驳回 ──→ [退回修改] → 重新提交
```

### 7.2 企业知识问答流程

```
[用户提问]
       │
       ▼
[意图识别] → 判断是否需要检索知识库
       │
       ▼
[混合检索] → 向量检索 + 关键词检索 → 召回相关文档段落
       │
       ▼
[重排序] → 按相关性排序，选取 Top-K 段落
       │
       ▼
[大模型生成] → 基于检索结果生成答案（RAG）
       │
       ▼
[答案输出 + 溯源引用] → 展示原文段落链接
```
